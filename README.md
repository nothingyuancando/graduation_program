# AI 个性化学习知识库

基于大语言模型的个性化学习知识库与复习路径生成系统。项目不再只是“AI 生成笔记”，而是把上传资料、手写笔记、关键知识点、概念双链、测验反馈、间隔复习和学习路径组织成一个面向学习闭环的工作台。

## 核心定位

- **Obsidian 式概念双链**：笔记正文支持 `[[概念]]` 语法，预览时会自动跳转到对应概念页。
- **Notion 式学习数据库**：笔记列表支持全部、按学科、待复习、薄弱点相关等多视图。
- **结构化学习对象**：笔记详情展示摘要、关键知识点、原文证据、相关概念、知识卡片和测验入口。
- **个性化反馈闭环**：测验提交后更新薄弱概念，复习卡片根据评分更新下次复习时间。
- **学习路径生成**：系统根据目标、已有笔记和 `weak_concepts` 生成更有针对性的学习计划。

## 功能概览

- 资料上传：支持文本、PDF、图片、URL、批量资料整合等入口。
- AI 分析：自动生成摘要、标签、关键知识点、原文证据、实体概念、卡片和结构化内容。
- 概念网络：提供 `/concepts` 概念总览和 `/concepts/[name]` 概念详情页，展示反向链接和相关笔记。
- 双链笔记：在正文中输入 `[[机器学习]]`，预览时会形成概念链接。
- 学习模板：新建笔记支持课程笔记、论文阅读、考试复习、错题整理和空白笔记。
- 多视图列表：笔记库支持按全部、学科、待复习、薄弱点相关切换。
- 今日复习：根据 `due_date` 找出到期卡片，并根据评分更新间隔。
- 测验反馈：自动生成测验，并把错误点写入学习画像。
- 学习路径：优先考虑薄弱概念，为目标生成分阶段复习计划。

## 快速开始

```bash
pnpm install
pnpm dev
```

启动后访问：

```text
http://localhost:5000
```

常用页面：

```text
/                 学习 Dashboard
/notes            学习资料库
/notes/new        模板化新建笔记
/concepts         概念网络
/learning-path    学习路径
/review           今日复习
/quiz/history     测验记录
```

## 环境变量

创建 `.env.local` 并配置数据库与至少一个大模型 Key：

```env
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_KEY=your-service-key

OPENAI_API_KEY=sk-your-key-here
```

项目也保留了 DeepSeek、通义千问、Kimi、智谱、豆包、SiliconFlow 等模型适配能力，按需配置对应 Key 即可。

## 毕设说明建议

推荐题目表述：

```text
基于大语言模型的个性化学习知识库与复习路径生成系统
```

系统创新点可以从三个角度展开：

- 从静态笔记转向“学习对象”：每篇笔记包含知识点、证据、概念、卡片和测验。
- 从单次生成转向“反馈闭环”：测验和复习会持续更新个人薄弱概念。
- 从文件管理转向“知识网络”：双链、概念页和反向链接帮助学生建立知识结构。
